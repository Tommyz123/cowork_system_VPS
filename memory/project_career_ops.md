**策略（2026-04-19主公确认）：** 作品敲门路线，不依赖投简历。

**核心动作：**
- 找1-2家认识的纽约大麻零售商，免费跑 Cannabis Budtender 3个月，积累案例和 testimonial
- 用结果敲门，而非简历

**目标公司：**
- EliseAI（垂直 AI Agent 方向）
- Dutchie / LeafLink（cannabis tech，主场优势）

**Why:** 主公36岁，目标进 AI 行业见识，非转型工程师；兼职/Contract/Freelance 优先；Cannabis Budtender 是最真实的作品，靠它敲门比投简历更有效。

**How to apply:** 讨论求职话题时，以此策略为基线，不建议去投传统 JD 或走大厂全职路线。路径见 `playbooks/career_ops.md`。

**进展（2026-05-28）：** 主公已在一家用 Alpine IQ（大麻零售 CRM/会员营销平台）的 dispensary 上班；老板（她）主动把该店 AIQ 的 **API key** 给主公，让他"分析点东西出来"帮生意——这正是策略里"做真实案例敲门"的机会。授权明确（老板主动给、目的就是分析）。注意：主公只有 API key、无 AIQ 后台账号；key 无法自行重置（需老板在后台重置）。下一步规划：先只读测试脚本验连通+权限，再做"客户洞察报告"（复购率/沉睡客户/VIP/品类偏好+拉回头客建议）。**How to apply:** 帮主公做这事 = 同时推进 P8 作品；纯分析无合规问题，仅"群发消息给客户"才触发 TCPA/大麻营销合规。

**薪资预期 + 切入策略（2026-06-02 确认）：**
- 定位"AI 相关入口 + 能当跳板 + 能学东西"，不追求一步到位，先上车再换座位。
- 薪资现实预期：第一份跳板 **$50-70k 或合同时薪**（初创/合同/AI 应用岗）；站稳 1 年后 **$80-110k**。注：平台统计的 entry-level $75-93k 多含经验/科班，主公够段时间才摸得到。
- 走"零经验自学转行靠作品集"路径，瞄准初创/合同/低门槛切入岗。求职平台：**Wellfound / Built In NYC / Dice**。
- 市场参考（2026-03 Glassdoor）：纽约 236 个 entry-level AI 岗，RAG/向量库技能最缺人，正好对口 P3。
**Why:** 主公零专业全职经验 + 英文口语瓶颈，需低门槛切入岗先积累履历。**How to apply:** 给求职建议时按此薪资预期和平台清单，不建议直接投高门槛/大厂岗。
