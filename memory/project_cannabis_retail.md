# P12 Cannabis Retail（2026 年核心主线）

**主线决策日期**：2026-05-14（基于八字真盘 + 行为层综合判断）

**三层定位**：
1. 拿 NY 大麻零售牌照 + 自营一家店作为实验场
2. 用 AI 把店内运营 / 顾客 / 营销系统化做差异化
3. 抽离 AI 模块 → SaaS 卖给 NY 100+ dispensary

**牌照状态**：申请中（CCB / OCM 流程），受 Organic Blooms v CCB（Index 904497-24）December queue 诉讼影响，每周一 09:00 EDT Discord 自动提醒查 NYSCEF

**团队结构**：主公自任"持牌方 + AI 总设计师"；雇 Assistant Manager 管员工日常；不雇 GM（AI 替代 GM 大部分职能）；6 个月硬截止把"亲自做"压到 30% 以下

**技术栈方向**：Dutchie POS / Flowhub / Treez 当基座（不自建 POS），主公做 AI 增强层（推荐 / 库存 / 营销 / 合规审计）

**核心功能模块**：QR + AI 推荐员 + EDDM + AI 独家折扣获客漏斗（EDDM 卡片 → QR → AI 对话 → 个性化推荐 → 独家折扣 → 下单 → 数据沉淀 → 复购）

**关联子模块（其他项目并入）**：
- P3 Cannabis Budtender → 店内 budtender AI 辅助
- P5 Legal Library / Organic Blooms 追踪 → 合规模块
- P1 marketing (Cannabis Advisor) → AI 增强层

**Why（命理 + 行为双重指向）**：
- 八字层（2026-06-03 真盘修正，详见 user_bazi.md）：**戊土日主·身强财旺·食神生财命**（非旧记的"丁火身弱"）；辛卯大运（33-42岁，2023-2032）；财星亥水当令一生有财路，命主线=食神生财（戊土→庚金才华→亥水财=做AI产品/SaaS变现）；2026 丙午年火土帮身过旺=易固执/项目过载，对治在"输出变现"非"再积累"
- 行为层：主公"系统化大脑+跨界整合+学习深度"配上 NY 大麻+AI 双拐点；但短板"项目过载+收尾迟疑+工程化倾向"——大麻零售作为主线恰好把 P1/P3/P5 整合，减少项目过载

**How to apply**：
- 主公提到大麻零售 / dispensary / NY 牌照 / AI 推荐员 / EDDM 等话题 → 自动路由到 `playbooks/cannabis_retail.md`
- 主公提到 P8 求职 / P9 交易 / P10 文件库时 → 提醒"主线已确定为 cannabis_retail，这条是否要冻结或并入"
- 主公讨论新项目时 → 先问"和 cannabis_retail 主线的关系"，避免新增分散
- 任何大麻相关决策要协助评估：① 是否服务主线 ② 是否符合 6 个月硬截止 ③ 是否符合"持牌方+AI 总设计师"分工
