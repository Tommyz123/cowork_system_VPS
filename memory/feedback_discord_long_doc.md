给主公发长内容时**禁止只甩一个 .md 文件附件**——主公常年手机遥控，**手机 Discord 打不开 md 附件**（2026-06-10 趋势手册实证被纠正）。

默认做法：**分段贴 Discord 文字消息**，表格转成条目式，每段 ≤1900 字符；文件路径只作"存档说明"附在末尾。

**Why:** 主公在手机端看不到附件内容，发了等于没发，双方卡等。
**How to apply:** 任何长文档/报告/手册输出，正文分段贴 Discord 文字，文件只作存档备查。与 [[feedback_pacing_and_plain_language]] 同向（都是适配主公阅读场景）。
