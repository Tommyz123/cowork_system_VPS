# INDEX 生成指令

> 这是一个通用模板文件。把它放进任何文件夹，Claude Code 读取后按以下要求生成该文件夹的 INDEX.md。
> 生成完成后，可以删除本文件，保留生成的 INDEX.md 即可。

---

## 📋 任务说明

你需要为当前文件夹生成一个 `INDEX.md` 索引文件。

**执行步骤：**
1. 扫描当前文件夹内所有文件和子文件夹
2. 打开并阅读每个文件的内容（代码、文档、数据等）
3. 按照下方格式生成 INDEX.md
4. 保存到当前文件夹根目录

---

## 📐 INDEX.md 标准格式

```markdown
# INDEX — [文件夹名称]

> 最后更新：[YYYY-MM-DD]
> 一句话描述：[这个文件夹是做什么的]
> 标签：#标签1 #标签2 #标签3

---

## 📁 子文件夹

### [子文件夹名]/
- **用途：** [这个子文件夹是干什么的]
- **关键内容：** [里面最重要的东西是什么]
- **标签：** #标签

---
```

## ✅ 写作要求

**文件清单的"关键内容"列必须写清楚：**
- Python 文件：写主要函数/类名和功能
- 数据库文件：写表名和关键字段
- 配置文件：写关键参数名
- 文档文件：写核心内容摘要
- 数据文件（CSV/JSON）：写字段名和数据量

**标签规则：**
- 用 `#` 开头，小写英文或中文均可
- 尽量具体，方便搜索（如 `#SQLite` 而不是 `#数据库`）
- 每个文件至少 2 个标签

**禁止：**
- 不要只写文件名，必须写内容
- 不要复制文件注释作为说明，要用自己的理解总结
- 不要遗漏任何文件（包括隐藏文件夹如 .claude/）
