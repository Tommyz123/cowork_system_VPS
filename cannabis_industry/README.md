# Cannabis Industry 行业信息库

> 大麻**行业通用信息**收集项目。与 Sage Seeds（单家公司项目）、cowork（个人系统）三者互不混合。

## 这是什么
收集整个纽约 / 美国大麻行业的通用信息：监管政策、市场数据、行业法律。
**不装任何单家公司（如 Sage Seeds）的营销、客户数据、API**——那些只在各自项目目录里。

## 边界（重要）
| 项目 | 装什么 | 不装什么 |
|------|--------|----------|
| **本项目** cannabis_industry | OCM 政策、监管动态、市场公开数据、行业法律 | 任何单家公司的私有数据 |
| Sage Seeds | 仅 Sage Seeds 一家的营销 / 客户数据分析 / API | 行业通用信息 |

→ 一句话：本项目是「整个行业」的文件夹，Sage Seeds 是「一家公司」的文件夹。

## 分层结构
| 目录 | 内容 | 状态 |
|------|------|------|
| `regulatory/` | OCM 政策、CCB 会议要点、监管动态 | ✅ 使用中 |
| `legal/` | 大麻法律资料 | 🔒 预留：等主公本地法律资料整理好后合并，现为空 |
| `market_data/` | 市场公开数据（销售额、牌照数等） | ⬜ 预留 |

## 信息来源
- OCM（NY Office of Cannabis Management）官网 / CCB 会议公开记录
- 公开新闻稿、市场报告

全为**公开信息**，无 API key / 无客户 PII → 直接进 cowork git 跟踪，无需物理隔离。
